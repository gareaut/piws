Population Imaging Web Service (PIWS)

The PIWS cube brings together imaging, genetics and questionnaire data into a database and publishes them using semantic web principles. A Metagen bioresource can be generated to serve genetic data: the proposed semantic web framework enables to bring together the chromosome, gene and snp information.
